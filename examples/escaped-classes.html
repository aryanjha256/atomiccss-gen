atomiccss-gen Example

Below is a demonstration of properly escaped CSS classes:

This box is styled with atomic CSS classes.

The class names in HTML are w-[50%] h-[200px] bg-[#eee], etc.

But in the generated CSS, they are properly escaped as .w-\[50\%\] .h-\[200px\] .bg-\[\#eee\], etc.

Generated CSS Example:


.w-\[50\%\] { width: 50%; }
.h-\[200px\] { height: 200px; }
.text-\[red\] { color: red; }
